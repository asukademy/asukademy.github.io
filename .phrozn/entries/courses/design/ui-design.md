User Interface 的概念最早來自人類與機器的互動，有多少概念是從傳統的機寫介面傳承而來？又有多少是網路時代出現的新型態設計呢？ 
本課程帶您從過去到未來，瞭解如何設計通用於多數使用者，並節省目標族群操作時間的優秀介面。
