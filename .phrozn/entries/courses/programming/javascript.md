Javascript 是網頁設計師與前端工程師的必備技能，掌握了這門技術，才有機會打造出流暢的互動性網站與網頁特效。Javascript 易於學習，
容易上手，但充滿許多陷阱，需要由有經驗的老師來帶領，適合程式基礎不高的設計師來學習。
