Laravel 是近年來發展最快速的 PHP MVC 框架之一，提供了簡潔的語法與高擴充性的架構，能夠輕鬆完成許多過去要耗費大量人力才能完成的網站功能。
我們將帶領有基礎 PHP 技能的學生進一步瞭解 MVC 框架的開發方式，以即團隊如何在框架中協同合作。

## 課程規劃

| 上課時間 | 時數 | 課程內容 |
| ------- | --- | ------- |
| - | 3hr | Laravel 導覽與安裝 |
| - | 3hr | Routing 與基礎 MVC 架構 |
| - | 3hr | CRUD 與 ORM |
| - | 3hr | 資料遷移與填充 |
| - | 3hr | View 與 Blade |
| - | 3hr | 表單與驗證 |
| - | 3hr | 會員認證 |
| - | 3hr | 網站安全性 |
